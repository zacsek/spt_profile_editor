The JSON files describing game objects (items, quests, traders, etc.) are located within your SPT-AKI server installation.

Assuming your SPT-AKI server is installed at `SERVER_PATH` (e.g., `C:\SPT-AKI\Server` on Windows or `/home/user/spt-aki/server` on Linux), you will typically find these files under the following structure:

`SERVER_PATH/Aki_Data/Server/database/`

Within this directory, the most relevant subfolders are:

*   **`templates`**: This folder contains many core game object definitions, including:
    *   `items.json`: Describes all in-game items.
    *   `quests.json`: Contains definitions for quests.
    *   `handbook.json`: Defines the item handbook structure.
    *   `globals.json`: Global game settings.
    *   `hideout.json`: Hideout area definitions.
    *   `locales.json`: Localization data (though often in subdirectories like `locales/global`).
*   **`traders`**: This folder contains JSON files for each trader, defining their inventory, loyalty levels, and other specific data.
*   **`bots`**: Contains AI bot configurations.
*   **`locations`**: Defines maps and their properties.

For example, the full path to `items.json` would typically be:
`SERVER_PATH/Aki_Data/Server/database/templates/items.json`

And for quests:
`SERVER_PATH/Aki_Data/Server/database/templates/quests.json`

You would need to replace `SERVER_PATH` with the actual path to your SPT-AKI server installation.
